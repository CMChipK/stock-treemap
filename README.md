# 股市族群熱力圖

這是一個展示股市族群（類股）表現的熱力圖網頁應用程式，使用 treemap 視覺化技術呈現各類股的相對大小和漲跌表現。

## 功能特色

- **熱力圖視覺化**：使用 d3.js 的 treemap 呈現各類股區塊
- **動態排序**：可切換以「成交量」或「資金流量絕對值」來決定區塊面積大小
- **顏色編碼**：區塊顏色深淺表示漲跌幅大小
  - 紅色：上漲（顏色越深漲幅越大）
  - 綠色：下跌（顏色越深跌幅越大）
- **互動功能**：點擊區塊可查看詳細資訊
- **深色主題**：符合現代 UI 設計的深色介面

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或使用本地伺服器（推薦）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需安裝 http-server)
   npx http-server -p 8000
   ```
3. 在瀏覽器中訪問 `http://localhost:8000`

## 資料結構

目前使用假資料，每個類股包含以下欄位：

```javascript
{
    name: '類股名稱',           // 例如：'半導體'
    category: '分類',          // 例如：'電子'
    changePercent: 2.87,       // 漲跌幅（%）
    fundsFlow: 548.33,         // 資金流向（億）
    volume: 1372.95,           // 成交量（億）
    stocks: ['2330', '2303']   // 該類股下的股票代號陣列
}
```
